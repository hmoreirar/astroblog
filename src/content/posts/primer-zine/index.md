# Manifiesto Digital

Este es el primer número de nuestro zine digital, donde exploramos la intersección entre el arte, la tecnología y el activismo.

## Arte y Código

El código no es solo una herramienta técnica, es una forma de expresión artística. Cada línea de código puede ser un trazo, cada algoritmo una composición.

## Hacktivismo

Reclamamos el derecho a usar la tecnología para cuestionar, subvertir y transformar las estructuras de poder.

## DIY

Porque hacerlo tú mismo no es solo una práctica, es una filosofía de autonomía y resistencia.
